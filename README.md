## Description

Playnest is a practice project focused on React hooks (useMemo, React.memo, useCallback, useReducer) and on structuring React components around simple game logic.

### Concepts and tools practiced

- **React Router DOM**: used to handle routing in the application. It allows navigation between different pages or views without reloading the whole page

- **useState hook**: used to create and manage local state inside my components, so the UI can update dynamically when the state changes

- **useEffect hook**: used to run game-related logic on component mount and to listen for state changes or side effects during the game

- **useReducer hook**: uused to group and manage all the game-related state in a centralized way, making it easier to handle different actions depending on the game logic.

- **useCallback**: used to memoize functions and prevent unnecessary re-creations on re-renders

- **React.memo**: used to prevent unnecessary re-renders of components when their props have not changed

- **custom Hooks** : used to create a reusable Timer shared across both games
